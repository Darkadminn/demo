# ООО «Обувь»
## Что из себя представляет

**ООО «Обувь»** - система для работы магазина по продаже обуви. Позволяет просматривать, добавлять, редактировать и удалять товары и заказы.

## Структура проекта


```
obyv010
├─ App.axaml
├─ App.axaml.cs
├─ Program.cs
├─ README.md
├─ app.manifest
├─ bin
├─ images
├─ models
├─ obj
├─ obyv010.csproj
├─ obyv010.sln
└─ views

```

## Система ролей

В системе реализована ролевая модель с 4 ролями.

**Администратор**
- Всё, что может менеджер
- Добавление, редактирование, удаление товаров
- Добавление, редактирование, удаление заказов

**Менеджер**
- Авторизация
- Просмотр товаров с фильтрацией, поиском и сортировкой
- Просмотр заказов

**Клиент**
- Авторизация
- Просмотр товаров

**Гость**
- Просмотр товаров
